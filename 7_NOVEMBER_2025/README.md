# Git Refresher

## Overview

Git is a distributed version control system that helps track changes in source code. It allows multiple developers to work on the same project efficiently while maintaining a complete history of changes.

## Key Points

- `git init` - Initialize a new repository
- `git clone` - Copy a remote repository locally
- `git add` - Stage changes for commit
- `git commit` - Save changes with a message
- `git push` and `git pull` - Sync local and remote repositories
- `git branch` - Create isolated development branches
- `git merge` - Combine branch changes
- `git log` and `git status` - Inspect repository state
