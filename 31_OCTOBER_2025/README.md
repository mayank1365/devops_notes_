# Linux Refresher

## Overview

Linux is an open source operating system widely used in servers, cloud platforms, and containers. It is the foundation of most DevOps tools and environments. Understanding Linux basics is essential for managing servers, troubleshooting issues, and automating tasks.

## Key Points

- **File System Structure**: Root, home, and directory hierarchy
- `ls -la` - List files including hidden files with permissions
- `cd` and `pwd` - Navigate directories
- `chmod` and `chown` - Manage file permissions and ownership
- `ps`, `top`, `htop` - Monitor running processes
- `df -h` and `du -sh` - Check disk usage
- `tail -f` - Monitor logs in real time
- **Pipes and Redirection**: Combine commands efficiently
